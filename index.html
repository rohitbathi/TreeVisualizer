Tree Visualizer

A one-page application for visualizing any binary tree.

How to use

  • Convert your tree object into a JSON string and copy the string.
  • Paste the string in the above textarea and hit the generate button.
  • If all the data you passed is correct, you will see the converted tree on the right.

JSON string format.

The JSON string must be like the example given below to ensure that the input binary tree is read correctly. 


{
    "nodeVal": 1,
    "left": {
        "nodeVal": 2,
        "left": null,
        "right": null
    },
    "right": {
        "nodeVal": 3,
        "left": {
        "nodeVal": 4,
        "left": null,
        "right": null
        },
        "right": null
    }
}
